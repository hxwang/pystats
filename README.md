#pystats

pystats is a python scripts for basic statistics tasks. pystats read from stand input by defualt, and use space as seperator.

### Usage
* Unix/Linux/Mac OS
  * `./pystats [options]`
  * Examples
    * `cat data.txt | ./pystats`
    * `./pystats < data.txt`
